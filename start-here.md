## 🗺️ The 30-second elevator pitch

SOMOSGAY is a 17-year-old LGBTQ+ rights + health NGO in Paraguay running the country's first LGBTQ+ community clinic. Their website (a minimal Next.js static export) undersells everything they do, doesn't support English-speaking donors, has no booking flow for the clinic, no app-store integration for Equis, no security headers, no analytics beyond UA-202057705-1, and no multilingual ES/EN/Guaraní switch. The repo you've opened contains the audit + benchmark against 7 comparable sites + a recommendations stack. Engagement model is undecided.

## 4. Cosas que el sitio NO debe tener

> Esta sección es CRÍTICA. Define restricciones explícitas.

- [x] ✅ **NO tracking de terceros invasivo** — no Facebook Pixel; no Google Analytics invasivo; solo Plausible privacy-friendly + GA4 con consent banner si es necesario
- [x] ✅ **NO capturar datos personales innecesarios** — especialmente en formularios de contacto
- [x] ✅ **NO usar imágenes de personas sin consentimiento explícito** — consent forms firmados
- [x] ✅ **NO afiliación religiosa** — org laica; verificado
- [x] ✅ **NO afiliación partidaria** — org apartidaria; verificado
- [x] ✅ **NO esconder el conflicto político** — parte de la marca (per Cazal + Paloma quotes)
- [x] ✅ **NO ocultar la historia 108** — memoria es identidad
- [ ] **NO usar la bandera de Paraguay sola sin contexto LGBT+** — reclaim, no absorb
- [ ] **NO templates genéricos de NGO internacional** — autenticidad paraguaya
- [ ] **NO asumir heterosexualidad del visitante** — default: LGBT+ friendly
- [ ] **NO carousel de donate popup** — agresivo; usar uno persistente al top + CTA en footer
- [ ] **NO autoplay video con audio** — accesibilidad + UX
- [ ] **NO fonts de Google** — CDN expone IPs a Google; self-host
- [ ] **NO Google Maps embed** — leaks IPs; usar OpenStreetMap
- [ ] **NO stock photos genéricas** — autenticidad importa
- [ ] **NO lenguaje condescendiente o pity-driven** — per Rosa Posa / AIREANA: "Don't be pitiful"
- [ ] **NO asumir heterosexualidad del visitante** (repetido para énfasis)

## 8. Donor journey (experiencia del donante)

8.1. **¿Qué ve un donante nuevo en el sitio?**

- 🟡 **Default recomendado:** Homepage → impacto stats → "Qué hacemos" → programas → donar (multi-canal) → gracias + newsletter signup; **flujo diseñado para conversión en 3 clicks**

8.2. **¿Qué ve un donante recurrente?**

- 🟡 **Default recomendado:** Homepage → "últimas noticias" → impact dashboard (MISMO donante puede ver SU historial si tiene login) → donar nuevamente; **promover recurring donations**

8.3. **¿Hay tiers de donación?**

- 🟡 **Default recomendado: Sí (3-4 tiers recomendado) — ej: $10 (amigo), $50 (aliado), $250 (sostén), $1,000+ (major gift); confirmar amounts exactos con cliente**

8.4. **¿Cómo reciben los donantes la confirmación de su donación?**

- 🟡 **Default recomendado: email inmediato vía Stripe + thank-you page con foto de impacto + opcional tax receipt si están habilitados a dar recibos deducibles (confirmar)**

8.5. **¿Cómo agradecen?**

- 🟡 **Default recomendado: thank-you email + newsletter mensual con impact updates + reconocimiento público (opcional) en página de supporters**

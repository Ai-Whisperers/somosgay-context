## 4. Cosas que el sitio NO debe tener

> Esta sección es CRÍTICA. Define restricciones explícitas.

- [ ] **NO tracking de terceros invasivo** (no Facebook Pixel, no Google Analytics invasivo) — solo Plausible privacy-friendly + GA4 con consent banner si es necesario
- [ ] **NO capturar datos personales innecesarios** (especialmente en formularios de contacto)
- [ ] **NO usar imágenes de personas sin consentimiento explícito** (consent forms firmados)
- [ ] **NO incluir contenido que exponga a comunidad a riesgo** en contexto político hostil
- [ ] **NO afiliación religiosa** (org laica; verificar directrices)
- [ ] **NO afiliación partidaria** (org apartidaria)
- [ ] **NO carousel de donate popup** (anunciante-agresivo)
- [ ] **NO autoplay video con audio** (accesibilidad + UX)
- [ ] **NO fonts de Google** (CDN expone IPs a Google; self-host)
- [ ] **NO Google Maps embed** (leaks IPs; usar OpenStreetMap)
- [ ] **NO stock photos genéricas** (autenticidad importa aquí)
- [ ] **NO lenguaje condescendiente o pity-driven** (per Rosa Posa / AIREANA: "Don't be pitiful")
- [ ] **NO esconder el conflicto político** (parte de la marca)
- [ ] **NO esconder la historia 108** (memoria es identidad)
- [ ] **NO usar la bandera de Paraguay sola sin contexto LGBT+** (reclaim, no absorb)
- [ ] **NO templates genéricos de NGO internacional** (autenticidad paraguaya)
- [ ] **NO asumir heterosexualidad del visitante** (default: LGBT+ friendly)

## TL;DR

The current site is **functional but undersells everything they do**. The biggest gaps:

1. **No service booking flow** — clinic appointment is WhatsApp-only
2. **No events calendar** — Pride, Besatón, talks all unlisted
3. **No patient stories** — no human face on the impact
4. **No app-store integration for Equis** — main utility is hidden
5. **No bilingual ES/EN switch** — international donors can't easily engage
6. **Deprecated analytics** — UA-202057705-1 (UA was sunset July 2023)
7. **No sitemap.xml** — bad for SEO
8. **Minimal photo library** — no program visuals
9. **No public impact dashboard** — no patient counts, no test counts
10. **News is the only current content** — `/news` has 168 pages, but no other living content
